# BlitzPress Core / Global Settings Prompt

Use the UI kit defined in `00-design-system-uikit.md` (Minimalist Executive UI Kit) and the shell from `01-admin-shell-navigation.md`.

## Goal

Design the **Core / Global Settings** page pattern for BlitzPress.

## Page objectives

The page should:
- feel like the authoritative, premium home for system-level configuration
- scale gracefully with clear typography hierarchy
- utilize the crisp form system with 4-8px radius inputs and soft backgrounds

## What to design

Create a core settings page with:
- prominent page header
- section navigation or tabs (active states in Electric Blue `#0052FF`)
- grouped settings cards/panels (white `#FFFFFF`, light shadow)
- strong label/help text hierarchy (`#141B2B` / `#565E71`)
- primary action buttons

## Deliverables

Provide:
- main core settings screen
- state variations
- responsive adaptation
